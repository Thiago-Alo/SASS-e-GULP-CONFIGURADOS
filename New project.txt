copiar o PACKAGE.json e o GULPFILE.js
- TERMINAL
1. npm install
2. gulp
3.1 Obs: o arquivo style.css será criado no passo 6, criar apenas os arquivos style.scss
3. Criar os arquivos INDEX, e as pastas CSS/SCSS e dentro da PASTA SCSS criar o arquivo STYLE.SCSS
4.1 Obs: o arquivo main.js será criado no passo 6, criar apenas os arquivos style.scss
4. Criar pasta JS/COMPONENTS e dentro da pasta COMPONENTS criar um arquivo teste.js - console.log('teste')
5. Obs: se não seguir os passos 3 e 4 ->
  ir no arquivo GULPFILE.js e atualizar os caminhos das pastas do CSS , do JS e dos WATCH
6. parar o gulp (CONTROL + C)
7. npm install -g sass
8. gulp
